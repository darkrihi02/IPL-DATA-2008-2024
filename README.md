# IPL-DATA-2008-2024
IPL Data Analysis (2008–2024) EDA of 260,920 IPL deliveries across 16 seasons using Python and Pandas. Dataset: IPL Complete Dataset 2008–2024 (Kaggle) . Tools: Python, Pandas, Matplotlib, Seaborn
Exploratory data analysis of 260,920 IPL deliveries across 16 seasons using Python and Pandas.
What I explored:

All-time run scorers and wicket takers
Team performance across seasons
Powerplay and death over specialists
MVP allrounders using custom criteria (200+ runs, 50+ wickets)
Toss impact on match results — found toss winners win only 50.6% of matches, barely above coin flip

Surprising finding: Toss advantage is a myth. Data across 1095 matches shows winning the toss gives almost no advantage.
