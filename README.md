# 🏏 IPL Data Analysis (2008–2024)

## 📌 Project Overview

This project presents an exploratory data analysis of the **Indian Premier League (IPL)** from **2008 to 2024** using Python and Pandas.

The analysis covers:

- **260,920+ ball-by-ball delivery records**
- **1,095 matches**
- **16 IPL seasons**

The goal of this project is to transform raw cricket data into meaningful insights about **player performance, team trends, match phases, all-round impact, and toss outcomes**.

---

## 🎯 Objectives

The project explores key analytical questions such as:

- Who are the leading run scorers in IPL history?
- Which bowlers have taken the most wickets?
- How has team performance changed across seasons?
- Which players perform best during the powerplay?
- Which players dominate the death overs?
- Who are the most impactful all-rounders?
- Does winning the toss significantly improve the chance of winning a match?

## 🔍 Analysis Performed

### 🏏 Batting Analysis
- Identified all-time leading run scorers
- Compared batting performance across seasons
- Analyzed player contribution using ball-by-ball data

### 🎯 Bowling Analysis
- Identified leading wicket takers
- Compared bowling performance across IPL seasons
- Evaluated wicket-taking impact

### ⚡ Match Phase Analysis
- Analyzed player performance during **Powerplay overs**
- Studied specialists in **Death overs**
- Compared performance across different phases of an innings

### 🌟 All-Rounder Analysis
Created custom criteria to identify impactful all-rounders:

- **200+ runs**
- **50+ wickets**

This helped identify players contributing significantly with both bat and ball.

### 🪙 Toss Impact Analysis
Analyzed whether winning the toss creates a meaningful match advantage.

**Key Finding:**

> Toss-winning teams won approximately **50.6% of matches**, suggesting that winning the toss provides almost no significant overall advantage.

The analysis was performed across **1,095 matches**.

---

## 💡 Key Insights

- IPL performance varies significantly across seasons and teams
- A small group of players consistently dominate long-term batting and bowling records
- Powerplay and death overs require different specialist skill sets
- All-rounders can be evaluated using combined batting and bowling contribution
- Toss advantage appears much weaker than commonly assumed
- Across 1,095 matches, toss winners won only around 50.6% of games
